
# Gitblit

---

- [Apache License Version 2.0](https://github.com/gitblit-org/gitblit#Apache-2.0-1-ov-file)
- In Java programmiert
<br>
![[Pasted image 20231214084542.png]]
- Läuft auf Windows Servern

---

- SSL Zertifikate werden automatisch generiert
- Eine Datei um Server und Gitblit selbst zu konfigurieren
- Einsetzbar als simpler Repository Viewer ohne Adminkontrolle/Benutzerkonten oder aber auch kompletter Git Stack mit clone, push etc. und Zugangskontrolle

---

- Alle Transportprotokolle werden unterstützt 
- Admins können alle Repositories und Accounts remote verwalten über WebUI/CMD/JSON RPC interface(?)
- [Viele weitere Features](http://www.gitblit.com/features.html)
